# Employee List App

View the list of all employees and view profile of an employee

# App architecture
- MVVM
- LiveData
- Room
- Retrofit
- Glide
- Master-Detail view

# Features

- View list of employees
- View profile of an employee
- Offline support
- Multilanguage support (English and Croatian)
- Event tracking (in progress)
